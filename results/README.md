# Results

- `metrics_baseline.csv` — zero-shot stage-1 CLIP (LAION ViT-L/14) on the Flickr8k
  1,000-image held-out test split, with 95% bootstrap confidence intervals.
- `summary.json` — the full run config, baseline aggregate metrics, and the LoRA
  ablation (zero-shot vs. fine-tuned CLIP-ViT-B/32, both directions, with paired
  bootstrap R@1 deltas and significance).
- `qualitative_examples.png` — sample text→image retrievals.
- `lora_adapter/` — the trained PEFT/LoRA adapter (r=8, q/k/v/out_proj, both
  towers) from the ablation in `summary.json`. Base model: `openai/clip-vit-base-patch32`.

`img_emb.npy`/`txt_emb.npy` (the raw embeddings) are regenerated by the notebook
and gitignored — rerun `notebooks/clip_retrieval_kaggle.ipynb` on Kaggle (GPU T4)
to reproduce everything above from scratch.
